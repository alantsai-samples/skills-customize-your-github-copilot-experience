
# 📘 Assignment: Hangman Game

## 🎯 Objective

Build a classic Hangman game in Python. Practice string manipulation, loops, conditionals, and random selection while creating an interactive word-guessing game.

## 📝 Tasks

### 🛠️ Implement Hangman Game Logic

#### Description
Write a Python program that allows a player to guess letters to reveal a hidden word. The game should track guesses and end when the word is guessed or the player runs out of attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Accept single-letter guesses from the user
- Display the current progress (e.g., _ a _ _ m a n)
- Track and display the number of incorrect guesses remaining
- End the game when the word is guessed or attempts are exhausted
- Show a win or lose message at the end

### 🛠️ Add User Experience Features

#### Description
Enhance your game with features that make it more user-friendly and fun.

#### Requirements
Completed program should:

- Prevent repeated guesses of the same letter
- Show all incorrect letters guessed so far
- Allow the player to play again without restarting the program

#### Example Output
```
Word: _ a _ _ m a n
Incorrect guesses left: 4
Incorrect letters: e, s, t
Guess a letter: 
```
